# Amazon Copy Asin
Chrome extension to easily copy ASIN's to clipboard from Amazon search results.

Supported marketplaces:
* www.amazon.com
* www.amazon.co.uk
* www.amazon.ca
* www.amazon.com.mx
* www.amazon.com.br
* www.amazon.de
* www.amazon.fr
* www.amazon.it
* www.amazon.es
* www.amazon.co.jp
* www.amazon.cn
* www.amazon.in
* www.amazon.com.au
* www.amazon.ae
